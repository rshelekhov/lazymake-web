
# Lazymake Landing Page

This is the landing page for [Lazymake](https://github.com/rshelekhov/lazymake) - a modern terminal user interface (TUI) for Makefiles that transforms how developers interact with Make.

## About Lazymake

Lazymake provides an interactive, visual browsing experience for Makefiles with powerful analytical tools including:

- **Dependency Visualization** - Interactive graphs showing target relationships and execution order
- **Variable Inspector** - Browse all Makefile variables in raw and expanded forms
- **Syntax Highlighting** - Automatic language detection for recipe code
- **Safety Detection** - Built-in safeguards for dangerous commands
- **Smart Search & History** - Per-project execution history with fuzzy filtering
- **Workspace Management** - Discover and manage multiple Makefiles
- **Performance Tracking** - Automatic execution timing to identify regressions

## Running the landing page

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
